Example: Circuit SystemY#

from opendssdirect import dss

# If you downloaded just the notebook, remember to update the path to the DSS script below

dss('Redirect "../../tests/data/13Bus/IEEE13Nodeckt.dss"')

The function dss.YMatrix.getYsparse() returns the vectors that form the compressed sparse representation of the matrix. This is the preferred usage for large system.

We recommend using scipy.sparse for this usage:

from scipy.sparse import csc_matrix

systemY = csc_matrix(dss.YMatrix.getYsparse())
